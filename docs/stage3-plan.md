# Stage 3: 사회 4-1 1단원 - 지도와 우리 지역

## 개요
- **교과**: 4학년 1학기 사회
- **단원**: 1단원 (소단원 1-1, 1-2 통합)
  - 1-1. 다양한 정보가 담긴 지도
  - 1-2. 우리 지역의 위치와 특성
- **게임 컨셉**: "지도 탐험대"
- **배경**: 호곡초등학교 주변 실제 지형 기반

## 스토리
마을의 지도가 찢어져 조각나 버렸다. 플레이어는 호곡초등학교에서 출발하여 마을 곳곳을 탐험하며 지도 조각을 모아 완성해야 한다.

## 맵 구성 (호곡초등학교 주변)

```
   [북서]          [북]          [북동]
 큰마을APT    탄현동성당    에듀포레푸르지오APT

[서]              [중앙]              [동]
산/저수지      호곡초등학교      호곡중학교
                                탄현근린공원

              [남]
            탄현역(지하철)
```

## 미션 흐름

| 단계 | 내용 |
|------|------|
| STEP 0 | 미션 시작 - 찢어진 지도 발견 |
| STEP 1 | 나침반 획득 - 방위 개념 학습 |
| STEP 2 | 지도 조각 4개 수집 - 각 장소 탐험 |
| STEP 3 | NPC 퀴즈 통과 - 지도/지역 관련 문제 |
| STEP 4 | 지도 완성 - 스테이지 클리어 |

## 퀴즈 (25문제)

### 1-1 다양한 정보가 담긴 지도 (12문제)
- 방위표(N/S/E/W), 방위표 없을 때 규칙
- 범례의 정의와 역할
- 축척 계산 (1:25,000)
- 지도 기호 (학교=門, 우체국=〒, 온천=♨)
- 큰 축척 vs 작은 축척 차이
- 등고선 (촘촘=급경사)
- 지도 색상 (파란색=물, 초록색=산/숲)

### 1-2 우리 지역의 위치와 특성 (13문제)
- 호곡초 기준 방위별 장소 위치 (6문제)
  - 북서: 큰마을아파트
  - 북동: 에듀포레푸르지오
  - 북: 탄현동성당
  - 동: 호곡중학교, 탄현근린공원
  - 서: 산, 저수지
  - 남: 탄현역
- 토지 이용 (주거지역, 공원, 교통시설)
- 자연환경과 주거환경의 공존
- 자연지물 vs 인문지물 구분
- 위치 설명 방법

```

- Walkable floor: bright/light colors (white, beige, light gray)
- Walls and obstacles: pure black (#000000)
- No gradients, no shadows, no anti-aliasing
- Flat color style, sharp pixel-perfect edges
- 마을 맵: 중앙에 학교, 북서 아파트, 북 성당, 북동 아파트, 동 중학교+공원, 서 산+저수지, 남 지하철역
```
